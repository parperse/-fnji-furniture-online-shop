2/5
椅背设计紧贴背部，给予脊椎足够支撑
-
椅背有张力，凸起设计可为腰部带来支撑，同时软包采用高品质杨木与桉木多层曲木板结合高回弹海绵，内置曲木板稍有弹性，让椅背可紧贴背部，配合实木搭脑，给予脊椎足够的支撑力，减轻身体负荷，配合实木搭脑，对背部有足够支撑，木纹连贯美观。
3/5
650mm扶手高度，坐姿舒适且易收纳
-
扶手高度650mm，为人呈坐姿时，手肘自然下垂的位置，易于身体放松，缓解久坐疲劳。不使用时，扶手以下部分可收至餐桌下，省空间。
4/5
整体线条流畅优美，保证观感完整性
-
椅腿均由一整块无暇木料，经多角度切割打磨而成，完整保留了木头的流畅纹理。扶手有棱，对接难度大，加工更难有误差，每个连接点都从不同维度去拼接，无论从设计到加工都需要手工打磨。坐垫上表层无拼接式包裹整片布料，保证观感完整性，相近色尼龙缝线经靠背处收于坐面后端,形成一条与木框架相互呼应的优美弧线，实现椅子的整体优美线条。
5/5
进口羊毛面料，触感柔软温暖
-
特别版坐面布料来自丹麦纺织品牌Kvardat羊毛量 90%。经线与纬线均由三种颜色纱线混合而成，不同角度与光线下呈现不同颜色，有浅灰、深紫两色可供选择；经典版坐面布料来自意大利，羊毛量 75%，面料触感柔软温暖，有柳黄、浅咖两色可供选择。

USING TIPS
1. 请勿将家具放置户外使用，尽量保持室内的干湿度稳定在35%-65%之间，避免长时间阳光暴晒；
2. 搬运或摆放家具时请勿大力拖拽；
3. 避免在家具表面覆盖潮湿或腐蚀性物品，避免用强效的清洁剂去污剂擦拭实木表面；
4. 椅类家具四脚可能略有不平，可用附赠的脚垫进行调节；
5. 此款家具软体部分不可拆洗，如被污损，请尽快用拧干的抹布或软体专用清洁剂清洁。也可以请专业的软体清洁公司上门养护。
